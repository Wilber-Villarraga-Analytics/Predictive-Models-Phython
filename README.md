# Modelado Predictivo y Ciencia de Datos con Python 

En este repositorio presento una serie de algoritmos aplicados a problemas de clasificación y optimización de datos, enfocados en la precisión y la interpretabilidad.

###  Modelos Implementados:
- **Regresión Logística:** Logro de una precisión del 100% en el conjunto de prueba tras un proceso riguroso de escalado de variables.
- **Árboles de Decisión:** Modelo con un 91% de confianza, optimizado para segmentar grupos con alta fidelidad.
- **KNN & Clustering:** Aplicación de K-Vecinos y Clustering Jerárquico para identificar perfiles y comportamientos.
- **PCA (Análisis de Componentes Principales):** Reducción de dimensiones logrando explicar el 96% de la varianza total, facilitando la visualización de datos complejos.

Cada modelo incluye un análisis de diagnóstico para asegurar que los resultados tengan rigor estadístico.
